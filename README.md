# Fakery - e-commerce

Fakeary is an e-commerce application, built with ReactJs. 
## Set up

You'll need Node.js on your local development machine.

1. Clone the repo:
```
git clone https://github.com/Malena-Ramirez/e-commerce.git
```

2. Install the dependencies

```
$ npm install
```

## Stack
- React
- Redux
- Firebase
- Styled Components
- Bootstrap
